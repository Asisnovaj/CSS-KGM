HTML terms

HTML
The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.
HTML element
An HTML element is a type of HTML (HyperText Markup Language) document component, one of several types of HTML nodes (there are also text nodes, comment nodes and others).
HTML tag
Elements are not tags. Some people refer to elements as tags (e.g., "the P tag"). Remember that the element is one thing, and the tag (be it start or end tag) is another.
angle bracket
An angle bracket or angle brace or angle cleat is an L-shaped fastener used to join two parts generally at a 90 degree angle.
HTML attribute
HTML attributes are special words used inside the opening tag to control the element's behaviour.
HTML command
An HTML element is a type of HTML (HyperText Markup Language) document component, one of several types of HTML nodes (there are also text nodes, comment nodes and others).
HTML object element

HTML special element
